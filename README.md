# QR Code Generator

A simple WPF desktop app to generate and customize QR codes.

## Features

- Generate QR codes from text or URLs
- Customize size, colors, and error correction level
- Save as PNG, JPG, or SVG
- Copy QR codes to clipboard

## Requirements

- .NET 10
- Windows 7+
